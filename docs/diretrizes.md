# Diretrizes

## Padroes Obrigatorios

<div class="grid grid-2" markdown>

<div class="card-accent" markdown>
### Deve Fazer

- Usar a paleta oficial definida no material de identidade.
- Aplicar a tipografia Metropolis quando disponivel.
- Manter composicoes limpas, objetivas e consistentes.
- Usar elementos graficos de vias e movimento como apoio, nao como ruído visual.
</div>

<div class="card-secondary" markdown>
### Deve Evitar

- Alterar cores sem validacao.
- Misturar tipografias sem criterio.
- Distorcer elementos graficos.
- Criar pecas que enfraquecam a leitura da marca.
</div>

</div>

## Boas Praticas

- Priorizar clareza da mensagem.
- Preservar contraste adequado entre texto e fundo.
- Usar elementos graficos para reforcar movimento e direcao.
- Validar materiais externos antes de publicacao.

## Checklist de Aplicacao

- [ ] A paleta esta correta.
- [ ] A tipografia esta coerente.
- [ ] Os elementos graficos estao proporcionais.
- [ ] A peca mantem comunicacao objetiva.
- [ ] A marca permanece legivel e reconhecivel.
