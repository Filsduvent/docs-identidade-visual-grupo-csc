# Identidade Visual Grupo CSC

Documentacao corporativa gerada pelo Corporate Docs Factory a partir do material de rebranding e identidade visual do Grupo CSC.

## Visualizacao Local

```bash
pip install mkdocs mkdocs-material
mkdocs serve
```

## Build

```bash
mkdocs build
```

## Deploy

O site e publicado no GitHub Pages pelo workflow em `.github/workflows/deploy.yml`.

Link esperado: https://Filsduvent.github.io/docs-identidade-visual-grupo-csc/

## Status

Conteudo gerado e preparado para publicacao em GitHub Pages.
